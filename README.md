# reversingLinkedLists
Playing with data structures with Typescript.


# Example
```
let list = new LinkedList();
list.push("hey");
list.push("hey2");
list.push("this is a linkedList");
list.traverse();
// list.pop();
// list.shift();
list.unshift("I'm the new head!");
list.traverse();
list.insert("I'm being inserted at position 2.", 2);
list.traverse();
list.remove(2);
list.traverse();
list.reverse();
list.traverse();

```
